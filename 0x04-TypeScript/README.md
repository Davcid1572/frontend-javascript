# 0x04-TypeScript

Short exercises and examples demonstrating TypeScript features for frontend development.

## Overview

This folder contains TypeScript practice files that cover basic to intermediate concepts: typing, interfaces, generics, modules, and DOM interaction examples tailored for the frontend-javascript course.

## Prerequisites

- Node.js (>= 14) on Windows
- npm
- TypeScript compiler (local or global)

## Setup

1. Open a terminal in this folder:
   - PowerShell / cmd: `cd c:\Users\IFESTECH\Desktop\frontend-javascript\0x04-TypeScript`
2. Install dependencies (if a package.json is present):
   - `npm install`

## Build & Run

- Compile TypeScript:
  - `npx tsc` (or `npm run build` if a build script exists)
- Run compiled code (example):
  - `node dist/index.js`

## Tests

If tests are present, run:

- `npm test`

## Structure

- src/ — TypeScript source files
- dist/ — compiled JavaScript output (generated)
- tests/ — unit tests (optional)
